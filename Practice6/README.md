# Practice 6: File Handling & Built-in Functions

This project covers the fundamentals of interacting with the file system and using Python's powerful built-in functional programming tools.

## Structure
- [cite_start]**file_handling/**: Basic CRUD operations (Read, Create, Update, Delete) on files.
- **directory_management/**: Creating, listing, and moving directories/files.
- **builtin_functions/**: Demonstrations of `map`, `filter`, `reduce`, `zip`, and `enumerate`.

## How to Run
Ensure you are in the root `Practice6/` directory.
Example:
```bash
python3 builtin_functions/map_filter_reduce.py
